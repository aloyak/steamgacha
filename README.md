# Steam Gacha

A gacha game for Steam games. Collect your library, discover new games, and have fun!

## Local Deployment

clone:

```sh
git clone https://github.com/aloyak/steamgacha.git
cd steamgacha
```

install dependencies:

```sh
npm install # or yarn install
```

start development server:

```sh
npm run dev # or yarn dev
```
